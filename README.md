# Laboratório PNETLab - Redes e Infraestrutura

Este repositório reúne laboratórios práticos desenvolvidos no PNETLab com foco em redes, infraestrutura corporativa e administração de ambientes Microsoft.

O objetivo é documentar cenários reais de estudo e simulação, utilizando equipamentos virtuais como MikroTik RouterOS, Windows Server, Windows Client e VPCS, aplicando conceitos de roteamento, segmentação de redes, serviços corporativos, segurança, NAT, VPN, monitoramento e boas práticas de infraestrutura.

## Objetivos do repositório

- Praticar redes e infraestrutura em ambiente virtualizado
- Criar cenários corporativos simulados
- Documentar topologias, planos de IP e configurações
- Estudar MikroTik RouterOS em cenários reais
- Implementar serviços Microsoft como Active Directory, DNS e DHCP
- Testar regras de firewall, NAT, VPN e segmentação de redes
- Evoluir tecnicamente por meio de laboratórios práticos

## Tecnologias utilizadas

## Estrutura dos laboratórios

Os laboratórios seguem uma evolução prática, começando por configurações básicas de rede e avançando para cenários corporativos mais completos.

Entre os principais temas abordados estão:

- Configuração inicial do MikroTik
- Segmentação de redes
- Roteamento entre sub-redes
- Serviços de domínio com Windows Server
- DHCP corporativo
- Regras de firewall
- NAT e publicação de serviços
- VPN
- Monitoramento e logs
- Boas práticas de documentação e operação

## Objetivo final

Construir um ambiente de laboratório completo, próximo de uma infraestrutura corporativa real, permitindo testes, estudos, documentação técnica e evolução prática em redes e infraestrutura.
